# 음성 알람 앱 (Voice Alarm App)

Expo로 만든 음성 알람 애플리케이션입니다. 날짜와 시간을 설정하고 메시지를 입력하면, 설정된 시간에 메시지를 반복해서 읽어주는 앱입니다.

## 주요 기능

### 1. 알람 설정
- **날짜 및 시간 선택**: 캘린더와 시간 선택기를 통해 정확한 시간 설정
- **메시지 입력**: 알람 시 재생될 메시지를 입력
- **알람 저장**: 설정된 알람을 기기에 저장

### 2. 알람 관리
- **알람 목록 보기**: 등록된 모든 알람을 한눈에 확인
- **알람 편집**: 기존 알람의 시간, 날짜, 메시지 수정 가능
- **알람 삭제**: 불필요한 알람 삭제
- **알람 활성화/비활성화**: 토글을 통한 알람 On/Off 제어

### 3. 알람 알림
- **백그라운드 동작**: 앱이 백그라운드에서도 알람이 울림
- **화면 잠금 상태**: 폰 화면이 꺼져 있어도 알림 기능 작동
- **노티피케이션**: 시스템 알림을 통한 시각적 및 음성 피드백
- **반복 재생**: 정지 버튼을 누르기 전까지 메시지를 5초 간격으로 반복

### 4. 알람 정지
- **정지 버튼**: 활성 알람을 즉시 정지 가능
- **쉬운 접근**: 화면 하단에서 쉽게 찾을 수 있는 정지 버튼

## 기술 스택

### 의존성
- **expo-notifications**: 시스템 알림 처리
- **expo-task-manager**: 백그라운드 작업 관리
- **expo-background-fetch**: 백그라운드 알람 실행
- **expo-av**: 오디오 재생
- **@react-native-async-storage/async-storage**: 로컬 데이터 저장
- **@react-native-community/datetimepicker**: 날짜/시간 선택

### 아키텍처
```
src/
├── app/                 # Expo Router 페이지
│   ├── (tabs)/
│   │   ├── index.tsx   # 홈 화면 (알람 목록)
│   │   └── _layout.tsx # 탭 네비게이션
│   └── _layout.tsx     # 루트 레이아웃
├── components/          # UI 컴포넌트
│   ├── AddEditAlarmScreen.tsx  # 알람 추가/편집
│   ├── AlarmListScreen.tsx     # 알람 목록
│   └── AlarmAlert.tsx          # 알람 활성 상태 표시
├── context/            # React Context
│   └── AlarmContext.tsx # 알람 상태 관리
├── utils/              # 유틸리티 함수
│   ├── alarmService.ts # 백그라운드 알람 서비스
│   └── alarmStorage.ts # 데이터 저장 및 조회
└── types/              # TypeScript 타입
    └── alarm.ts        # 알람 인터페이스
```

## 필요한 권한

### iOS
- **마이크 권한** (NSMicrophoneUsageDescription)
- **백그라운드 오디오 재생**

### Android
- **알람 정확 시간 설정** (SCHEDULE_EXACT_ALARM)
- **음성 녹음** (RECORD_AUDIO)
- **알림** (POST_NOTIFICATIONS)

## 사용 방법

### 1. 새 알람 추가
1. 홈 화면의 `+` 버튼 클릭
2. 알람 메시지 입력
3. 날짜 선택
4. 시간 선택
5. `저장` 버튼 클릭

### 2. 알람 활성화
- 추가한 알람 옆의 토글 스위치를 On 상태로 설정

### 3. 알람 편집
- 알람 목록에서 수정하고 싶은 알람의 `편집` 버튼 클릭
- 내용 수정 후 `저장` 클릭

### 4. 알람 정지
- 알람이 울릴 때 화면에 나타나는 `정지` 버튼 클릭
- 또는 알람 목록에서 빨간 정지 아이콘 클릭

## 주의사항

1. **배터리 최적화**: 장치의 배터리 최적화로 인해 백그라운드 작업이 제한될 수 있으므로, 앱을 배터리 최적화 제외 목록에 추가해주세요.

2. **권한 설정**: 각 플랫폼에서 필요한 권한을 허용해야 모든 기능이 제대로 작동합니다.

3. **시스템 시간**: 알람은 기기의 시스템 시간을 기준으로 작동합니다.

4. **지속성**: 음성 재생은 5초 간격으로 반복되며, 사용자가 정지 버튼을 눌 때까지 계속됩니다.

## 문제 해결

### 백그라운드 알람이 작동하지 않는 경우
1. 앱 권한 확인
2. 배터리 최적화 설정 확인
3. 시스템 시간 확인
4. 앱 재시작

### 알림이 표시되지 않는 경우
1. 알림 권한 허용 확인
2. 기기의 알림 설정 확인
3. 앱 재설치
